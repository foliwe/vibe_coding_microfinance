# Build Roadmap

## Phase 1: Foundation

- create monorepo structure
- lock shared types and docs
- define schema and metrics

## Phase 2: Backend Core

- implement schema and RLS
- add ledger workflow
- add approval and loan logic

## Phase 3: Admin Panel

- build dashboards
- build branch/member/agent/loan forms
- build approvals, reports, and reconciliation

## Phase 4: Mobile App

- build role-based flows
- build agent field workflows
- build member account and loan views

## Phase 5: Integration And Hardening

- wire live backend data
- add fraud controls
- add exports and acceptance scenarios

## Phase 6: Release Readiness

- deployment docs
- backup/restore
- final QA and training
