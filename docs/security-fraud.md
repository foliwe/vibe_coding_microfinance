# Security And Fraud Controls

## Authentication

- first-login password reset
- staff transaction PIN
- device binding for staff accounts
- member app relock with PIN or biometric

## Transaction safeguards

- maker-checker approval
- per-transaction and per-day limits
- agent cash drawer tracking
- branch reconciliation workflow
- no destructive edits to approved ledger data

## Fraud signals

- duplicate transaction within a short time window
- abnormal offline delay before sync
- suspicious withdrawal frequency
- out-of-pattern loan rate or amount
- agent acting outside assigned branch/member scope
- reconciliation variances

## Operational controls

- remote device/session revocation
- audit logging for all sensitive actions
- document upload trail for identity and collateral
- branch-level exception reports
